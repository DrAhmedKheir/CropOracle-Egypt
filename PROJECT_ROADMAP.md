# Project Roadmap

## Phase 1 — Prototype

- Prepare Flask webhook.
- Connect WhatsApp sandbox.
- Run local multi-agent demo.
- Prepare one APSIM wheat template.
- Test with one location such as Kafr El-Sheikh.

## Phase 2 — Scientific validation

- Calibrate wheat cultivar parameters.
- Validate APSIM outputs against observed Egyptian wheat trials.
- Build soil profiles for Delta, Middle Egypt, Upper Egypt, and new lands.
- Compare outputs with historical yield statistics.

## Phase 3 — Decision-support expansion

- Add irrigation scenarios.
- Add nitrogen fertilizer scenarios.
- Add heat-stress and water-stress indicators.
- Add Arabic message templates.
- Add dashboard for researchers and administrators.

## Phase 4 — National scaling

- Precompute simulations by governorate or grid cell.
- Add Redis/Celery workers.
- Add PostgreSQL database.
- Add farmer profile memory.
- Add voice message support.
- Launch pilot with selected farmers or extension officers.

## Phase 5 — Research outputs

Potential outputs:

- Research paper on WhatsApp-based crop simulation advisory for Egypt.
- Conference presentation on multi-agent AI and APSIM integration.
- Decision-support platform for wheat climate adaptation.
- Extension manual for digital advisory deployment.
